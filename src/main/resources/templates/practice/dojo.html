Level 0

問題

Eメールアドレスはローカル部とドメインを「@」を繋いだ文字列で表されます。

ローカル部を s ,ドメインを t として、文字列がそれぞれ変数に格納されています。

以下の構文に沿った文字列を出力してください。

入力例1

String local = "jouhou";
String domain = "hcs.ac.jp";

出力例1

jouhou@hcs.ac.jp

※結果は、標準出力(コンソール)へ出力すること※

Level 1

問題

とある町Aでは、新しくテーマパークができました。

とある町Aにはもともと100人の人が住んでいます。

A 村では人口が毎月 X 人ずつ増加します。Y 月後の A 村の人口はいくらですか？

入力

X Y
  • 1文字目に A 村が毎月増加する人数 X が与えられます。
  • 2文字目に何か月後の人口を求めるかを表す Y が与えられます。
  • 1文字目と2文字目の間に半角スペースが挿入されます。

入力例1

180 2

出力例1

460

入力例2

300 11

出力例2

3400

Level 3

問題

以下のルールで文字変換を行います。

  1. とある文字列から母音を取り除いて子音のみを連結する
  2. ここで母音とは "a", "e", "i", "o", "u" の 5 つのアルファベットの小文字( "a", "e", "i", "o", "u" )、大文字( "A", "E", "I", "O", "U" )を指し、子音とはそれ以外のアルファベットを意味する。

入力例1

Torvalds

出力例1

Trvlds

入力例2

Hokkaido

出力例2

Hkkd

Level 4

問題

以下のルールで○✕ゲームが行われています。

  1. 縦 1 マス、横 5 マスのマス目を作る。
  2. 全てのマスに o または x を書き込む。
  3. o が横に連続して 3 つ以上並んでいるなら、 o の勝ちである。
  4. そうでなくて、 x が横に連続して 3 つ以上並んでいるなら、 x の勝ちである。
  5. このどちらでもない場合、引き分けである。

○✕ゲームの結果を判定してください。

入力例1

xooox

出力例1

o

o が連続して 3 つ並んでいるので、 o の勝ちです。

入力例2

xxxxx

出力例2

x

x が連続して 5 つ並んでいるので、 x の勝ちです。

入力例3

xoxxo

出力例3

draw

o も x も 3 つ以上連続して並んでいないため、引き分けです。

入力値
